# API Documentation

## Categories
* `POST /categories  `
* `GET /categories ` 

## Products
* `POST /products ` 
* `GET /products  `

## Variants
* `POST /variants  `
* `GET /variants/:productId  `

## Pricing
* `GET /products/:product_id/price?quantity=& user_tier=`

## Cart
* `POST /cart/items`
* `PUT /cart/items`
* `DELETE /cart/:cart_item_id`

## Checkout
* `POST /cart/checkout`
